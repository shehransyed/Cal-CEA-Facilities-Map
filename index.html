California Controlled Environment Agriculture (CEA) Facilities

This interactive map displays information about Controlled Environment Agriculture (CEA) facilities across California.
Use the filters to explore facilities by type and crop, and click markers on the map for more details.

If you are a grower or facility operator, you can submit a request to add your facility to the map by clicking the "Add Facility" button: Add Facility
To correct or update an existing listing, please email snsyed@ucdavis.edu.

This is a work-in-progress; more features and visual enhancements will be added. Stay tuned!
